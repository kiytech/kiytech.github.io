# PianoRemote について

「PianoRemote」は、カワイデジタルピアノ専用のリモートアプリです。音色を変更したり、内蔵曲を楽しんだりすることができます[*1]。

# ご自宅のピアノをスマホ・タブレットでらくらく操作

ピアノの様々な機能を「PianoRemote」から簡単に操作できます。

# ピアノ本体タッチパネルと同じUIデザイン

「PianoRemote」は、スマホやタブレットからでも使いやすいようにデザインされています。 全ての UI が共通化されているため、ピアノ本体とアプリ、それぞれの使い方を覚える必要がありません。

# 音楽再生アプリ感覚で使える音楽プレイヤー

カワイデジタルピアノの内蔵曲を多数収録[*2]。一般的な音楽再生アプリを使う感覚で、音楽再生を楽しむことができます。 ピアノ音色だけでなく、オーケストラやチェレスタ等の音色に変更することで、雰囲気を変えて楽しむことができます。

# 聴く以外の発見がある「レッスン」・「コンサートマジック」

内蔵楽曲を聴くだけでなく、練習に便利な「レッスン」モードや指一本でお手軽に演奏ができる「コンサートマジック」モード[*3] によって、新たな感覚で楽曲を楽しむことができます。「コンサートマジック」モードは、リズムをとりながら楽曲を覚えるのにも便利です。

# 豊富な内蔵機能をコントロール

練習に欠かせない「メトロノーム」、ピアノ調律師のようにピアノの音をお好みに調整できる「コンサートチューナー」、音色を組み合わせてデジタルピアノならではの演奏を楽しめる「デュアル・スプリット」等、豊富な内蔵機能を「PianoRemote」だけで操作できます[*4]。

# Bluetooth MIDIでかんたん接続

無線通信「Bluetooth MIDI」に対応。ケーブルを別途用意することなくすぐに接続して楽しめます[*5]。

# iOS/Android、スマートフォン/タブレット対応

「PianoRemote」は、iOS/Android、スマートフォン/タブレット、様々なデバイスに対応。お持ちのデバイスを買い替えても安心、長くお使い頂けます[*6]。

*1 カワイデジタルピアノとスマートフォンまたはタブレットを接続する必要があります。
*2 ピアノ本体内蔵曲の内、著作権のある曲は収録されておりません。
*3 楽曲データの構成上「レッスン」、「コンサートマジック」モードに対応しない曲があります。
*4 「レコーダー」、「USBメモリプレイヤー」機能のコントロールは対応していません。
*5 別売のUSBケーブルやBluetoothMIDIアダプター「Quicco Sound mi.1」での接続にも対応しています。
*6 iOS11またはAndroid7.0以上 / OS標準のMIDI機能が搭載されたデバイスが必要です。対応状況については、各デバイスメーカーへお問い合わせください。

# 対応機種 ：カワイデジタルピアノ CA99/CA79/CA9900GP
